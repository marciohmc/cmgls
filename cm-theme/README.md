# C&M Global Services - Tema WordPress Customizado

Tema desenvolvido para WordPress utilizando **Tailwind CSS** e **PHP**. Focado em alta performance e design tech-moderno.

## Tecnologias
- WordPress v6.x
- Tailwind CSS (via JIT CDN)
- Google Fonts (Space Grotesk & Inter)

## Instalação
1. Baixe a pasta `cm-theme`.
2. Compacte em ZIP.
3. No painel WordPress, vá em Aparência > Temas > Adicionar Novo > Enviar Tema.
4. Ative o tema.
